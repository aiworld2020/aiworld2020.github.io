Introduction to Data Science

Data Science uses scientific methods, processes, algorithms and systems to extract knowledge and insights from noisy, structured and unstructured data. This class introduces to Machine Learning, Data Analysis, Data Visualization, Jupyter Notebooks, Pandas, Numpy and Scikit-learn. You will be provided with sources and references to continue the learning after the class.

Prerequisites:

Students should be familiar with basic programming concepts in Python or Java before enrolling in this class.

Day 1

  • Introduction to Numpy
  • Arrays Slicing
  • Arrays Indexing
  • Array Shape
  • Array Split
  • Array Sort

Day 2

  • Introduction to Pandas
  • Pandas DataFrames
  • Pandas Series
  • Read CSV
  • Read JSON
  • Analyzing Data

Day 3

  • Project - Spotify Song Recommendations (Part 1)
  • Download and Visualize Data
  • Features of Songs
  • Scatter Plots

Day 4

  • Project - Spotify Song Recommendations (Part 2)
  • Minimal Distance
  • Quantify Similarity with Multiple features
  • Data Normalization
  • Cosine Similarity
