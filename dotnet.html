Develop .Net Applications

This class is designed for absolute beginners. Students will learn to use Visual Studio and Unity programming environment. Students will develop algorithms using programming language C# and Unity Software. Every

Resources:

  • Unity https://unity.com/ (Download Free Personal Version 2019.1.4 for yourself before the class) This includes the Visual studio package.
  • Microsoft Visual Studio 2019 (Download 2019 version, scroll to bottom, click "older downloads", then scroll until you see 2017, select the community version (v 15.9), make sure it has ConsoleApp.NET framework as an option when you open it, use x64) https://visualstudio.microsoft.com/downloads/ (You do not need to download this if you already downloaded Unity, but if you are unable to download unity for any reason, then please download this.)
  • Web-based compiler in case of technical difficulties: https://replit.com/

Day 1

  • Hello World in C#
  • Data Types, Variables, Literals, Operators
  • Keywords, Switch Statements
  • Break, Continue, Goto, Return and Throw
  • Loops in C#

Day 2

  • Class and Object
  • Constructors
  • Inheritance
  • Encapsulation
  • Abstraction
  • Project 1. Fortune Teller

Day 3

  • Methods
  • Method Overloading
  • Method Overriding
  • Method Hiding
  • String Functions
  • Project 2. Whack a Mole

Day 4

  • Project 2. Window Form Game
  • Variable declaration/assignment
  • Conditional statements
  • Random number generation
  • Functions
  • Timer
  • Basic window form controls (button/label/textbox/picturebox)
