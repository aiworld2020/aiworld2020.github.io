Learn Web Design

In this course students will learn about the web design, HTML, CSS and BootStrap. Students will also learn how to launch their own website, domain registeration, creating a website (like this one) using github pages.

Prerequisites:

No prior coding experience is required. Students should be able to use computer independently for a better learning experience.

Day 1

  • HTML
  • Basic Tags
  • Tables
  • Images
  • Lists
  • Links

Day 2

  • CSS
  • Inline, Embedded and External CSS
  • Basic CSS Selectors
  • Hierarchy
  • Attribute Selector and Properties

Day 3

  • Bootstrap
  • Setup
  • Grid System
  • Components - Labels, Buttons, Forms
  • Input Group
  • Navbar

Day 4

  • Website
  • Domain Registeration
  • Github Pages
  • Jekyll Markdown
  • CName Changes
